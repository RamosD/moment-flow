# `pages/` — Route screens

One folder per route. A page composes widgets/features/entities/shared and is
mounted by the router (FE-004).

Slices: `dashboard`, `campaigns`, `campaign-detail`, `campaign-war-room`,
`not-found` (implemented). The rest are implemented in FE-009/FE-011.

May import from `widgets`, `features`, `entities` and `shared`.
